- v1.0 - **Phase 4: Progression — Levels, Achievements & Prestige.** Persistent player profile (`localStorage`, survives run restarts), skeleton ported from a sister project's (RMUV) leveling/achievement/prestige system — stripped of everything that was really about RMUV's 40-game arcade catalog (avatars, cosmetics, genre shelves, cross-game score normalization); this game has one CP source, so it needed none of that. A data-driven achievement registry (`achievements.js`, 21 badges across common/special/epic/legendary) is tested via a fixpoint loop on every event so one unlock can cascade into another the same tick. Prestige resets the CP climb at an escalating level gate (10, 20, 30…) for a permanent +10%/prestige CP-gain bonus plus spendable skill points, sunk into a 4-node skill tree (Gold/Damage/Build/Fortification Mastery) that stacks with the existing Command Core bonuses.
