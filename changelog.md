- v1.0 - **Phase 4: Progression — Levels, Achievements & Prestige.** Persistent player profile (`localStorage`, survives run restarts), skeleton ported from a sister project's leveling/achievement/prestige system — stripped of everything that was really about the sister project's 40-game arcade catalog (avatars, cosmetics, genre shelves, cross-game score normalization); this game has one CP source, so it needed none of that.
